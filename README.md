# ERPNext Project Dashboard

A modern React-based project management dashboard integrated with ERPNext REST APIs.

## Features
- Dashboard with analytics
- Kanban board (drag & drop tasks)
- Project management
- ERPNext API integration

## Setup

Create `.env` file:

VITE_API_BASE_URL=http://localhost:8000

VITE_API_KEY=your_key
VITE_API_SECRET=your_secret

Install dependencies:
npm install

Run:
npm run dev
